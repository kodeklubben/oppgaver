# Sammendrag

Oppgaven skal gi trening i bruk av enkle strengfunksjoner, samt litt bruk av
lister, løkker og dictionaries.

## Forkunnskapskrav:

Strengfunksjoner:

* `.join()`
* `.replace()`
* `.lower()`
* `.count()`

Løkker:

* `for i in range(start, stop, step)`
* `for element in lst`

Lister:

*  `list(text)`
* `.remove()`
* `.append()`
* `lst[index]`

Dictionaries:

* `dct[key] = value`
* `value = dct[key]`
