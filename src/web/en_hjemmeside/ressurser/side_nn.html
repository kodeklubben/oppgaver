Gruppere ting

Eg har sett dette avsnittet i ein boks. <div> er ein boks du kan setje andre ting inn i. Den er nyttig å bruke når du vil gruppere ting saman.

Du kan bruke klasser til å skilje like element i koden frå kvarandre. Det kan vere nyttig viss du vil bruke same type taggar til litt ulike ting. Klasser er ikkje synlege, men etter kvart vil du lære korleis du kan bruke dei til å endre utsjånaden på ting på sida.

Du kan ha mange element med same klasse.

Du kan bruke id til nesten det same: bruk id når du vil skilje ut eit heilt spesielt element.

Du kan berre ha eitt element med den same ID-en, men som ein bonus kan du lenke til det elementet på sida.
