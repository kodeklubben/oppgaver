# Om oppgaven {.activity}

I denne oppgaven skal elevene få ulike bilder til å vises på micro:bit-skjermen.

## Oppgaven passer til: {.check}

__Fag__: Programmering

__Anbefalte trinn__: 5. trinn - VG3

__Tema__: Programmering.

__Tidsbruk__: Dobbelttime

## Kompetansemål {.challenge}

- [ ] __Programmering, 10. trinn:__ bruke flere programmeringsspråk der minst ett
  er tekstbasert

## Forslag til læringsmål {.challenge}

- [ ] Elevene kan bruke kode til å vise bilder på skjermen.

## Forslag til vurderingskriterier {.challenge}

- [ ] Oppgaven er grunnleggende, og kan ikke brukes alene for vurdering av
  kompetansemålet.

## Forutsetninger og utstyr {.challenge}

- [ ] __Forutsetninger__: Oppgaven er en introduksjon til micro:bit, og krever
  ingen forkunnskaper eller erfaring.

- [ ] __Utstyr__: Datamaskin med tilgang til Internett, micro:bit og
  micro-usb-kabel.
